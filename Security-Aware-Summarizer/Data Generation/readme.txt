Data Collection Folder has the raw prompts from various dataset and our extracted harmful keywords from gpt-4 and also the reference summaries generated from Llama-3.1 70B.

The data mix folder contains the prompt and question aggregation of harmful, harmless and ambiguous prompts and questions. Also contains the mixture of prompts and questions and our final dataset for finetuning and evaluation.

All output csv files will be found at: https://huggingface.co/datasets/Shagoto/Data-Generation-Summary/tree/main/After%20Data%20Generation%20Data
